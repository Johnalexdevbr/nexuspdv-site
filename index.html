Dúvidas Frequentes

Perguntas & Respostas

O sistema funciona sem internet?
Sim! O NexusPDV funciona localmente. A internet é necessária apenas para backup em nuvem e IA.
Para quais sistemas operacionais funciona?
Compatível com Windows 10 e 11 (64-bit). O instalador já inclui o Java Runtime — não precisa instalar nada antes.
Posso usar em mais de um computador?
O plano padrão cobre uma instalação. Para múltiplos caixas ou filiais, entre em contato para uma proposta personalizada.
Como funciona o Programa de Parceiros?
Você se cadastra, recebe treinamento gratuito, indica o sistema para comerciantes da sua região e ganha comissão recorrente por cada cliente ativo. Sem investimento inicial.
O sistema emite NFC-e?
Sim! Módulo fiscal completo com NFC-e, transmissão para SEFAZ, QR Code e certificado digital A1.
Quanto tempo leva para ativar?
Após confirmação, instalado e configurado em até 24 horas com treinamento incluído.
