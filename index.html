Perguntas Frequentes

O sistema funciona sem internet?
Sim! Funciona localmente no seu computador. Internet só para backup em nuvem e IA.
Quais sistemas operacionais são compatíveis?
Windows 10 e 11 (64-bit). O instalador já inclui o Java Runtime — não precisa instalar nada.
Posso usar em mais de um computador?
O plano padrão cobre uma instalação. Para múltiplos caixas ou filiais, entre em contato.
Como funciona o Programa de Parceiros?
Você se cadastra, recebe treinamento gratuito, indica o sistema e ganha comissão recorrente. Sem investimento inicial.
O sistema emite NFC-e?
Sim! Módulo fiscal completo com NFC-e, transmissão para SEFAZ, QR Code e certificado A1.
Quanto tempo para ativar?
Até 24 horas após confirmação, com instalação e treinamento incluídos.
